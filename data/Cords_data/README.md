# Processed raw data files for lung cancer dataset

The orginal data was obtained from the original Cords et al. 2024 publication 
[Cancer-associated fibroblast phenotypes are associated with patient outcome in non-small cell lung cancer](https://www.cell.com/cancer-cell/fulltext/S1535-6108(23)00449-X).

The file raw_data.tar.gz is a compressed version of the folder containing the processed data files for 1534 images. 

To build the graph objects and also for the later steps of the method, it is needed to extract the folder from the compressed version:
```
tar -xvzf raw_data.tar.gz
```
